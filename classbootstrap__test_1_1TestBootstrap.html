Attempt to bootstrap two nodes at once, to assert the second bootstrapped node fails, and does not interfere.

Start a one node cluster and run a stress write workload. Start up a second node, and wait for the first node to detect it has joined the cluster. While the second node is bootstrapping, start a third node. This should fail.
