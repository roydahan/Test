def datahelp.row_describes_data (   row)

Returns True if this appears to be a row describing data, otherwise False.

Meant to be used in conjunction with filter to prune out those rows that don't actually describe data, such as empty strings or decorations that delimit headers from actual data (i.e. '+-—|-—|--—+')

Creates db rows using given session, with table name provided, using data formatted like:

|colname1|colname2| +-----—+-----—+ |value2 |value2 |

format_funcs should be a dictionary of {columnname: function} if data needs to be formatted before being included in CQL.

Returns a list of maps describing the data created.
