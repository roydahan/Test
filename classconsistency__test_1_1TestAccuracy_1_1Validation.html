def validate_users (   self)

First validation function: update the users table sending different values to different sessions and check that when strong_consistency is true (R + W > N) we read back the latest value from all sessions.

If strong_consistency is false we instead check that we read back the latest value from at least the number of nodes we wrote to.

def validate_counters (   self)

Second validation function: update the counters table sending different values to different sessions and check that when strong_consistency is true (R + W > N) we read back the latest value from all sessions.

If strong_consistency is false we instead check that we read back the latest value from at least the number of nodes we wrote to.
