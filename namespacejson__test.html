Takes an instance of dtest.Tester (or a subclass), completes some basic setup, and returns a dict of "globs" to be used as a doctest execution context.

This provides a doctest environment to test cql via python driver, or cqlsh directly. Cqlsh commands made via the cqlsh_* commands are sent (almost) exactly as provided, but a keyspace will be injected before commands for convenience.

If you prefer to customize setup, pass prepare=False and provide your own cql connection object and a list of ccm nodes. When nodes are provided, the first node will be used for connections and cqlsh.
