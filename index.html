Scylla Test Plans

This is Scylla Test Plan documentation system version 2! do we really save the comments?.
The followings pages contain a test plan document per release and a MASTER test plan that is the quality target for ScyllaDB.

The MASTER Test Plan will be used as the testing "Wish list" and each release test plan will aim to cover more parts of it based on their priority.

The Release Test Plans should cover all test cases and scenarios that should be run for this release and its exit criteria.
